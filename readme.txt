Responsive Social Share is the best social media plugin for WordPress that integrates share buttons to your website.

== Description ==

Responsive Social Share is a light-weight WordPress plugin that adds Social Share Buttons onto your blog posts, pages, and other post types. The floating ability allows this eye-catching social media bar to get you maximum shares.

= Easy to Use =

Responsive Social Share is extremely easy to use. The simple options interface allows you to drag-and-drop the social buttons that you want to display. You can also use the drag-drop interface to control the order of how social share buttons appear on your site.

== Installation ==

= Manual Installation =

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

= Inside WordPress =

1. From the Admin -> Plugins page, click 'Add New'.
2. Search for 'Responsive Social Share', then click 'Install Now'.
3. Once installed, click the 'Active Plugin' link.
